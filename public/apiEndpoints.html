About the Project

BotBuilder is an All-In-One (A.I.O) Discord bot designed to provide server owners with a comprehensive suite of features to manage and enhance their Discord servers. Whether you're running a small community or a large server, BotBuilder offers tools and functionalities to streamline your server management. Our primary goal is to support the Minecraft community by offering features that help server owners create their dream servers in a matter of weeks. From custom commands to making your own A.P.I, BotBuilder is here to make your server management experience seamless and efficient.

How to Use the API

Basic Steps

  1. Authorization: Include your API key in the Authorization header
  2. Endpoint: Use the appropriate endpoint for your action
  3. Request Method: Use the correct HTTP method (GET, POST, PATCH, DELETE)
  4. Request Body: Include JSON data when required

Example Request

{
  "method": "POST",
  "headers": {
    "Authorization": "YOUR_API_KEY"
  },
  "body": {
    "key": "value"
  }
}

Example Response

{
  "success": true,
  "message": "Request was successful",
  "data": {
    "key": "value"
  }
}
